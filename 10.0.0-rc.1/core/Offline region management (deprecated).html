  • MBMOfflineRegionGeometryDefinition

    Deprecated

    An offline region definition is a geographic region defined by a style URL, a geometry, zoom range, and device pixel ratio. Both minZoom and maxZoom must be ≥ 0, and maxZoom must be ≥ minZoom. maxZoom may be ∞, in which case for each tile source, the region will include tiles from minZoom up to the maximum zoom level provided by that source. pixelRatio must be ≥ 0 and should typically be 1.0 or 2.0.

    Deprecated

    A region’s status includes its active/inactive state as well as counts of the number of resources that have completed downloading, their total size in bytes, and the total number of resources that are required.
